第三章公式模板：

```
3.1 MaxPressure 基础：
    p_i(t) = Σ_{a∈A_i} q_a(t) - Σ_{b∈B_i} q_b(t)
    选择相位 i* = argmax_i p_i(t)

3.2 容量归一化：
    p_i^norm(t) = p_i(t) / C_i
    其中 C_i 为相位 i 对应车道的存储容量

3.3 溢出门控：
    if occupancy(edge) > 0.9:
        force_green(edge)  # 强制放行，防止锁死

3.5 EWMA 预测：
    predicted = α × current + (1-α) × last_predicted
```
