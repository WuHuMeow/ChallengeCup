# W6 任务书：交付 B（DB）

> 周期：8/24（周日）- 8/31（周六）
> 核心目标：视频最终修改、提交材料中的视频准备、答辩支持

---

## 每日任务

### Day 1（8/24 周日）
1. 参加全员 review 会议
2. 记录视频的修改意见（节奏、画面、字幕、旁白）

### Day 2（8/25 周一）
1. 视频最终修改：
   - 根据 review 意见调整
   - 修复画面/字幕问题
   - 确认旁白与画面对齐
   - 确认总时长 5-8 分钟（不超过 8 分钟）
2. 重新渲染最终版

### Day 3（8/26 周二）
1. 视频质量最终检查：
   - 分辨率 1920×1080 ✓
   - 格式 MP4 (H.264) ✓
   - 大小 < 500MB ✓
   - 音画同步 ✓
   - 字幕无错别字 ✓
2. 输出到 `demo/output/final_demo_v2.mp4`
3. 发给 TL 最终确认

### Day 4（8/27 周三）
1. 准备视频提交版本：
   - 确认比赛平台对视频格式/大小的要求
   - 如果需要压缩，用 ffmpeg 调整码率
   - 准备视频封面图（第一帧截图）
2. 确认所有图表文件在仓库中

### Day 5（8/28 周四）
1. 整理 `demo/` 目录最终状态：
   - output/final_demo.mp4（最终视频）
   - 视频脚本.md（存档）
   - assets/（素材存档）
2. 删除过大的中间文件（保留最终版即可）

### Day 6（8/29 周五）
1. 参加模拟答辩
2. 负责答辩中的视频播放环节：
   - 确认播放设备兼容
   - 准备备用 U 盘
   - 确认音量/投影效果
3. 如果答辩需要现场演示系统，准备演示环境

### Day 7（8/30-8/31）
1. **8/30**：协助最终提交（确认视频上传成功）
2. **8/31**：Buffer
