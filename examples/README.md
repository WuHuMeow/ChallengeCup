# examples/

## 模块职责

存放最小可运行示例，帮助 teammates 快速理解如何运行单次仿真和批量实验。

## 当前完成情况

- [x] `run_fixed_time.py`：运行单个路口固定配时基线仿真的示例脚本。

## 待完成情况

- [ ] `run_ca_maxpressure.py`：CA-MP 算法示例。
- [ ] `run_actuated.py`：感应控制算法示例。
- [ ] `run_batch.py`：多场景多算法批量跑批示例。

## 需求分析

| 需求 | 说明 |
|------|------|
| 快速上手 | 新队员通过示例脚本 1 分钟跑通第一个仿真 |
| 算法对比 | 提供同一场景下三种算法运行的示例 |
| 批量实验 | 提供 360 次全量跑批的示例 |

## 关键文件

| 文件 | 说明 |
|------|------|
| `run_fixed_time.py` | 固定配时基线示例 |

## 使用方式

```bash
python examples/run_fixed_time.py 1
```
